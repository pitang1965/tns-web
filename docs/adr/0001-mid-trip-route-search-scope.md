# ADR-0001: 途中からのルート検索の範囲を日程内に限定する

## ステータス

承認済み

## コンテキスト

「途中からのルート検索」（ここから最後までのルート検索）機能を追加するにあたり、「最後まで」の範囲を次のいずれかにする必要があった。

- **案A**: その日の日程の終わりまで
- **案B**: 旅程全体の終わりまで（日またぎ）

## 決定

**案A（日程内に限定）** を採用する。

## 理由

- 既存の「全体ルート検索」も日程単位で動作しており、ユーザーモデルが一貫する
- 旅行中に翌日以降のルートを含めた案内を行うユースケースは現実的でない（宿泊をまたいだ連続ドライブは想定しない）
- 日をまたぐ場合、Googleマップへ渡せる経由地の上限（10か所）を超えるリスクが高まる

## トレードオフ

複数日をまとめてルート確認したいユーザーには対応できない。その場合は既存の日程ごとの「全体ルート検索」を順番に使うことで代替できる。
